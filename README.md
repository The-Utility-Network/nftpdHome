## üöÄ Getting Started

To run the Protocol Explorer locally:

1.  **Clone the repository**
    ```bash
    git clone https://github.com/TUC-Ledger1/nftpd.git
    cd nftpd
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Ignite the daemon**
    ```bash
    npm run dev
    ```

4.  **Access the interface**
    Open [http://localhost:3000](http://localhost:3000) to view:
    -   `/documents` - Protocol Whitepaper
    -   `/our-model` - Philosophical Treatise
